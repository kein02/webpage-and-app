# 增肌助手 · 居家健身

一款专为居家训练设计的增肌健身 App，零依赖、纯前端、离线可用。

## 核心特点

- **膝盖友好**：所有 23 个动作均为低冲击/无跳跃版本，适合膝盖有伤的训练者
- **一周 5 练**：胸+三头、背+二头、肩部+核心、臀+腿、全身代谢，每次约 30 分钟
- **无需器材**：全部动作仅需自重、椅子、书包等居家物品
- **详细指导**：每个动作包含分步指导、呼吸节奏、目标肌群、要点提示、常见错误、进阶方向和替代动作

## 功能模块

| 模块 | 说明 |
|------|------|
| 📋 计划 | 根据星期自动匹配当日训练计划，展示本周安排 |
| 🏋️ 训练 | 逐动作逐组引导，组间休息计时器，实时次数调节 |
| 📖 动作库 | 23 个动作按肌群分类，支持搜索和筛选，详情页含个人记录 |
| 📊 进度 | 总训练次数/时长/容量统计，近 4 周柱状图 |
| ⚙️ 设置 | 个人信息、营养参考（自动计算）、数据管理 |

## 技术栈

- 纯 HTML + CSS + JavaScript，无框架无构建工具
- IndexedDB 本地持久化存储
- 暗色主题移动端适配
