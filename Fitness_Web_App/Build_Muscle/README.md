# 增肌助手 · 居家健身

一款专为居家训练设计的增肌健身 Web App，零依赖、纯前端、离线可用，手机浏览器打开即用。

## 核心特点

- **70+ 健身动作**：涵盖徒手、哑铃（水瓶替代）、弹力带三类，无需桌椅等家具依赖
- **双保护模式**：膝盖保护（过滤跳跃/冲击动作）+ 手腕保护（过滤手掌撑地动作），开启后计划自动替换为安全动作
- **一周 5 练**：胸+三头、背+二头、肩部+核心、臀+腿、全身代谢，每次约 30 分钟
- **详细指导**：每个动作包含分步指导、呼吸节奏、目标肌群、要点提示、常见错误、进阶方向和替代动作
- **实时训练引导**：逐动作逐组引导，组间休息倒计时 + 语音播报，训练数据本地记录

## 功能模块

| 模块 | 说明 |
|------|------|
| 📋 计划 | 根据星期自动匹配当日训练计划，展示本周安排 |
| 🏋️ 训练 | 逐动作逐组引导，组间休息计时器，实时次数调节 |
| 📖 动作库 | 70+ 动作按肌群分类，支持关键词搜索和筛选，详情页含个人记录 |
| 📊 进度 | 总训练次数/时长/容量统计，近 4 周柱状图 |
| ⚙️ 设置 | 个人信息、营养参考（自动计算）、膝盖/手腕保护模式开关、数据管理 |

## 技术栈

- 纯 HTML + CSS + JavaScript，无框架无构建工具
- IndexedDB 本地持久化存储
- 暗色主题移动端适配
- GitHub Pages 部署

## 快速开始

1. 浏览器打开：https://kein02.github.io/webpage-and-app/Build_Muscle/
2. 填写个人资料 → 开始今日训练
