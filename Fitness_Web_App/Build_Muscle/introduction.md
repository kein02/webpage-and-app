# 增肌助手 · 使用说明

居家徒手健身，科学增肌。无需器械，打开就能练。

---

## 一、打开应用

用浏览器打开增肌助手：

> **GitHub Pages**：https://kein02.github.io/webpage-and-app/Build_Muscle/

也可以将网页"添加到主屏幕"（安装为 PWA），之后即使没有网络也能打开使用。

---

## 二、底部导航栏

应用底部有五个标签页，从左到右依次是：

| 图标 | 标签 | 作用 |
|------|------|------|
| 📋 | **计划** | 查看今日训练计划，一键开始 |
| 🏋️ | **训练中** | 训练进行中时显示，包含组数、次数、休息倒计时 |
| 📖 | **动作库** | 浏览全部动作，可按肌群分类筛选，也可搜索 |
| 📊 | **进度** | 查看累计训练次数、时长、容量和每周趋势图 |
| ⚙️ | **设置** | 个人信息、保护模式、语音开关、数据管理 |

## 三、训练流程

### 1. 查看今日计划

打开「计划」页，顶部卡片显示**今天的训练内容**，包括：

- 训练主题（如"胸 + 三头"、"背 + 二头"等）
- 每个动作的名称、组数、次数、休息时间
- 本周七天计划预览

### 2. 开始训练

点击任意动作卡片或底部的 **"▶ 开始训练"** 按钮，进入训练界面。

训练界面包含：

- **进度条** — 显示当前做到第几个动作（如 2/5）
- **动作卡片** — 当前动作名称、目标组数/次数、休息秒数
- **次数调节器** — 点击 − / + 按钮调整实际完成的次数（默认等于目标次数）
- **完成按钮** — "完成这组"（中间组）或 "完成动作"（最后一组）
- **导航按钮** — "← 上一个" / "下一个 →" 可在动作间跳转
- **结束训练** — "⏹ 结束训练" 可随时终止本次训练

### 3. 组间休息

完成一组后，会自动进入休息倒计时：

- 屏幕中央出现全屏计时器，从设定的秒数倒数
- 语音/蜂鸣提示休息开始
- 每 10 秒播报一次，最后 10 秒每秒播报
- 休息结束自动进入下一组
- 可以点击 **"跳过休息"** 直接进入

### 4. 完成训练

做到最后一个动作的最后一组时，点击 **"完成动作"** 即可结束训练。应用会自动记录本次训练的时长、总容量等信息。

## 四、动作库

在「动作库」页面：

- **搜索** — 顶部搜索框输入动作名称即可过滤
- **分类筛选** — 点击肌群标签（胸、背、肩、臂、核心、臀、全身）按类别查看
- **查看详情** — 点击任意动作卡片，进入详情页，包含：
  - 所需器材
  - 呼吸节奏
  - 目标肌群
  - 动作说明
  - 分步指导（1-7 步）
  - 要点提示
  - 常见错误
  - 进阶方向
  - 替代动作
  - 最近 30 天训练记录迷你图表

---

## 五、进度追踪

在「进度」页面可以看到：

- **总训练次数** — 累计完成了多少次训练
- **总训练分钟** — 累计训练时长
- **总容量** — 累计完成的组数 × 次数的总和
- **近 4 周柱状图** — 每周训练次数的趋势
- **最近训练列表** — 最近 7 次的训练日期、计划名称、用时

每次训练结束后自动跳转到此页查看结果。

## 六、设置

在「设置」页面可以调整：

### 个人信息

- **体重 (kg)** — 默认 70，用于营养参考计算
- **身高 (cm)** — 默认 170
- **年龄** — 默认 25

### 训练设置

- **组间休息 (秒)** — 默认 60 秒，可按动作自定义
- **难度级别** — 初级 / 中级 / 进阶

### 保护模式

- **膝盖保护** — 开启后只显示对膝盖友好的动作（所有动作默认 `knee: true`）
- **手腕保护** — 开启后只显示对手腕友好的动作（排除手掌撑地类如标准俯卧撑、钻石俯卧撑）

> 💡 如果手腕或膝盖有伤，建议开启对应的保护模式，应用会自动替换为更安全的动作。

### 语音设置

- **语音播报开关** — 训练时自动播报动作信息和倒计时
- **测试语音** — 点击按钮测试蜂鸣是否正常

> 本应用使用蜂鸣音（不同频率和次数区分场景）而非文字语音，兼容性更好，老旧设备也能正常使用。

### 营养参考

设置体重后自动计算：

- **每日推荐热量** = 体重(kg) × 35 + 400 大卡
- **每日推荐蛋白质** = 体重(kg) × 1.8 克

### 数据管理

- **清除所有数据** — 删除训练记录、进度数据和设置（需二次确认）

## 七、一周训练计划

| 星期 | 主题 | 侧重肌群 |
|------|------|----------|
| 周一 | 胸 + 三头 | 胸大肌、肱三头肌 |
| 周二 | 背 + 二头 | 背阔肌、肱二头肌 |
| 周三 | 肩部 + 核心 | 三角肌、腹直肌/腹斜肌 |
| 周四 | 臀 + 腿(髋主导) | 臀大肌、腘绳肌 |
| 周五 | 全身代谢 | 全身燃脂 + 心肺 |
| 周末 | 休息 | 恢复 |

每天训练约 30 分钟，每个动作 3-4 组，组间休息 30-60 秒。

## 八、常见问题

**Q: 为什么没有声音？**
A: 检查手机音量是否打开。点击设置页的"🔊 点击测试语音"按钮排查。部分浏览器需要首次触摸页面后才能播放声音，打开应用后随便点一下即可。

**Q: 可以离线使用吗？**
A: 可以。安装为 PWA 后（添加到主屏幕），即使没有网络也能打开和使用，所有数据保存在本地。

**Q: 数据会丢失吗？**
A: 不会。所有数据存储在浏览器本地（IndexedDB），清除浏览器数据或删除应用才会丢失。如需备份，建议定期截图进度页面。

**Q: 如何更换训练计划？**
A: 应用根据当天星期几自动切换计划。你也可以在「计划」页点击任意动作单独开始该动作的训练。

**Q: 手腕疼怎么办？**
A: 去「设置」开启"手腕保护模式"，应用会替换掉手掌撑地的动作（如俯卧撑），改为使用哑铃、弹力带或等长收缩类动作。

## 九、技术信息

- **纯前端应用** — 无服务器，无后端，所有数据存在本地
- **数据存储** — IndexedDB（浏览器内置数据库）
- **PWA** — 支持添加到主屏幕，离线可用
- **Service Worker** — 自动缓存资源，更新时会提示刷新
- **无构建工具** — 纯 HTML + CSS + JavaScript
